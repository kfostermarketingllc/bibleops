# Bible Study Curriculum Generator - AI Agent Library

## Overview

This is a specialized AI-powered Bible study curriculum generator featuring **14 expert agents**, each fine-tuned for specific aspects of biblical education and spiritual formation.

Based on the proven CurrForge curriculum model, this system creates comprehensive, theologically sound, denominationally-sensitive Bible study materials in minutes.

**New in v3.0**: AWS S3 integration with temporary download links (14-day expiration) for better email deliverability.

## Original 11 Bible Study Agents

### **1. Foundational Materials & Reference Agent**
**Purpose**: Establishes the pedagogical and hermeneutical framework for the entire curriculum

**Based on 12 Essential Resources**:
- How to Read the Bible For All Its Worth (Fee & Stuart)
- The Joy of Discovery in Bible Study (Wald)
- Rick Warren's Bible Study Methods (Warren)
- Women of the Word (Wilkin)
- Creative Bible Teaching (Richards & Bredfeldt)
- Understanding Bible by Design (Lester)
- Design for Teaching and Training (Ford)
- The Art of Teaching the Bible (Blair)
- Essential Bible Study Tools for Ministry (Bauer)
- Writing for Life and Ministry
- 16 Bible Studies for Your Small Group (Lokkesmoe)

**Output**: 3-5 page foundational framework that guides all other agents

---

### **2. Bible Version Agent**
**Purpose**: Recommends appropriate Bible translation(s) based on study goals, denomination, and audience

**Expertise**:
- Translation philosophies (word-for-word vs. thought-for-thought)
- Denominational preferences (KJV, NIV, ESV, NASB, NLT, etc.)
- Readability levels
- Textual basis (Textus Receptus vs. Critical Text)

**Output**: 2-3 page translation recommendation with rationale

---

### **3. Denominational Theology Agent**
**Purpose**: Applies appropriate theological framework based on denominational tradition

**Covers**:
- Catholic & Orthodox traditions
- Protestant mainline (Lutheran, Presbyterian, Anglican, Methodist)
- Protestant evangelical (Baptist, Pentecostal, Non-denominational)
- Distinctive doctrines and interpretive approaches
- Ecumenical sensitivity

**Output**: 3-4 page theological framework respecting denominational distinctives

---

### **4. Biblical Context Agent**
**Purpose**: Provides historical, cultural, and geographical background

**Includes**:
- Historical background (authorship, date, setting)
- Cultural context (customs, social structures)
- Geographical setting (maps, locations, archaeology)
- Literary context (book structure, surrounding passages)
- Canonical context (redemptive history)

**Output**: 4-5 page context document with maps, timelines, and charts

---

### **5. Hermeneutics & Interpretation Agent**
**Purpose**: Applies proper interpretive principles based on genre

**Covers**:
- Genre-specific interpretation (narrative, poetry, prophecy, epistle, etc.)
- Grammatical-historical method
- Literal, allegorical, typological interpretation
- Common interpretive challenges
- Observation ‚Üí Interpretation ‚Üí Application process

**Output**: 4-5 page hermeneutical guide with practical examples

---

### **6. Original Languages Agent**
**Purpose**: Provides accessible Hebrew/Greek word studies and textual insights

**Includes**:
- Key word studies with transliteration
- Grammatical insights (verb tenses, particles, etc.)
- Literary and rhetorical features
- Textual variants and manuscript evidence
- Translation comparisons

**Output**: 3-4 page guide with accessible language insights for non-specialists

---

### **7. Cross-Reference & Theology Agent**
**Purpose**: Connects passage to broader biblical themes and systematic theology

**Covers**:
- Primary cross-references (10-15 key passages)
- Biblical theology connections (covenants, kingdom, redemptive history)
- Systematic theology analysis
- Christocentric interpretation
- Thematic connections across Scripture
- Quotations and allusions

**Output**: 4-5 page guide showing how passage fits in whole counsel of God

- Personal, relational, and missional application

- Concrete action steps
- Life transformation focus

## üéØ How It Works

### Input Required:
- **Scripture passage/book** OR **Book to study** (with title, author, ISBN optional)
- **Target audience** (age, spiritual maturity)
- **Denomination** (for theological sensitivity)
- **Study duration** (weeks/months)
- **Bible version preference**
- **Study objectives**
- **Special considerations** (group size, teaching context, etc.)

### Process:
1. **Book Research Agent** (if book study) - analyzes the book
2. **Foundational Agent runs NEXT** - establishes framework
3. **All other agents** run in parallel receiving context from steps 1-2
4. **Each agent generates** specialized content
5. **PDFs created** for each component (up to 14 PDFs)
6. **Complete curriculum** emailed to you ready for use

### Time:
**6-10 minutes** for complete generation (13-14 AI agent calls depending on study type)

## üöÄ Setup & Deployment

### Prerequisites:
- Node.js 18+ and npm
- Anthropic API key
- Mailchimp Transactional API key
- AWS account with S3 access

### Quick Start:

```bash
# Clone repository
git clone https://github.com/yourusername/bibleops.git
cd bibleops

# Install dependencies
npm install

# Configure environment (.env file)
ANTHROPIC_API_KEY=your_anthropic_key
MAILCHIMP_API_KEY=your_mailchimp_key
AWS_ACCESS_KEY_ID=your_aws_key
AWS_SECRET_ACCESS_KEY=your_aws_secret
AWS_REGION=us-east-1
AWS_S3_BUCKET_NAME=bibleops-pdfs

# Start server
npm start
```

### AWS S3 Setup:

See detailed instructions in **`AWS_S3_SETUP.md`** for:
- Creating S3 bucket
- Setting up lifecycle policy (14-day auto-delete)
- Creating IAM user with credentials
- Configuring environment variables

**Quick Start**: See **`QUICK_START_S3.md`** for 5-minute setup

### Deployment (Render.com):

1. Connect GitHub repository to Render
2. Add environment variables in dashboard
